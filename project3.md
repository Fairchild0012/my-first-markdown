# PDF Converter

A simple tool to convert Word documents into PDF.

## Installation

1. Download the project files.  
2. Open with Microsoft Word. 3. Save as PDF.  

## Features

- Easy Word to PDF conversion  
- Works on PC and mobile  
- No internet needed
